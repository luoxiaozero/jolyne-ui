# Props

### JoAnchor

| 名称     | 类型               | 默认值      | 描述             |
| -------- | ------------------ | ----------- | ---------------- |
| listenTo | `[Object, String]` | `undefined` | 监听元素的滚动条 |

### JoAnchorLink

| 名称  | 类型     | 默认值      | 描述     |
| ----- | -------- | ----------- | -------- |
| title | `String` | `undefined` | 标题     |
| href  | `String` | `undefined` | 定位链接 |
