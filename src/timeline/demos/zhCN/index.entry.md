# 时间线 Timeline

## 演示

```demo
basis
horizontal
```

## Timeline Props

| 名称       | 类型      | 默认值  | 描述         |
| ---------- | --------- | ------- | ------------ |
| horizontal | `Boolean` | `false` | 是否水平显示 |

## TimelineItem Props

| 名称  | 类型                                                       | 默认值      | 描述             |
| ----- | ---------------------------------------------------------- | ----------- | ---------------- |
| type  | `"default" \| "info" \| "success" \| "warning" \| "error"` | `"default"` | 标签类型         |
| title | `String`                                                   | `undefined` | 标题             |
| time  | `String`                                                   | `undefined` | 时间或者当副标题 |
