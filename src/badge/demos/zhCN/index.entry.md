# Props

| 名称     | 类型                                                       | 默认值      | 描述             |
| -------- | ---------------------------------------------------------- | ----------- | ---------------- |
| type     | `"default" \| "error" \| "info" \| "success" \| "warning"` | `default`   | 标记类型         |
| value    | `Number`                                                   | `undefined` | 标记数量         |
| maxValue | `Number`                                                   | `undefined` | 标记最大数量     |
| dot      | `Boolean`                                                  | `false`     | 是否显示标记为点 |
