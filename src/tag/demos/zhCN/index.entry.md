## Props

| 名称     | 类型                                                       | 默认值      | 描述       |
| -------- | ---------------------------------------------------------- | ----------- | ---------- |
| type     | `"default" \| "info" \| "success" \| "warning" \| "error"` | `"default"` | 标签类型   |
| closable | `boolean`                                                  | `false`     | 是否可关闭 |
| onClose  | `(e: MouseEvent) => void`                                  | `undefined` | 关闭事件   |
