## Props

| 名称           | 类型                   | 默认值      | 描述               |
| -------------- | ---------------------- | ----------- | ------------------ |
| accept         | `String`               | `undefined` | 接受的文件类型     |
| multiple       | `Boolean`              | `false`     | 是否支持多个文件   |
| onBeforeUpload | `(file: File) => void` | `undefined` | 文件上传之前的回调 |
