# Props

| 名称     | 类型                         | 默认值    | 描述                                                          |
| -------- | ---------------------------- | --------- | ------------------------------------------------------------- |
| vertical | `Boolean`                    | `false`   | 是否垂直排列                                                  |
| justify  | `"start" \| "end"`           | `"start"` | 水平排列方式                                                  |
| wrap     | `Boolean`                    | `true`    | 是否超出换行                                                  |
| size     | `Number \| [Number, Number]` | `8`       | 为数字时，是垂直和水平间距；为数组时，是 [水平间距, 垂直间距] |
