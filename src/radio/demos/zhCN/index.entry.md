## Radio Props

| 名称    | 类型      | 默认值      | 描述                   |
| ------- | --------- | ----------- | ---------------------- |
| checked | `Boolean` | `false`     | 是否选中               |
| name    | `String`  | `undefined` | radio 元素的 name 属性 |
| value    | `String`  | `undefined` | radio 元素的 value 属性 |

## RadioGroup Props

| 名称  | 类型     | 默认值      | 描述                              |
| ----- | -------- | ----------- | --------------------------------- |
| value | `String` | `undefined` | 选中的值                          |
| name  | `String` | `undefined` | 选项组内部 radio 元素的 name 属性 |
