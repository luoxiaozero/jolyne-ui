## Props

| 名称     | 类型      | 默认值      | 描述       |
| -------- | --------- | ----------- | ---------- |
| width    | `String`  | `undefined` | 一行的宽度 |
| height   | `String`  | `undefined` | 一行的高度 |
| repeat   | `Number`  | `1`         | 行数       |
| animated | `Boolean` | `true`      | 动画       |
