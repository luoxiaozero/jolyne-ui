# Props

### Layout Props 和 LayoutContent Props

| 名称     | 类型                     | 默认值     | 描述           |
| -------- | ------------------------ | ---------- | -------------- |
| hasSider | `Boolean`                | `false`    | 是否存在侧边栏 |
| position | `"static" \| "absolute"` | `"static"` | 布局定位模式   |

### LayoutHeader Props

| 名称     | 类型      | 默认值  | 描述         |
| -------- | --------- | ------- | ------------ |
| bordered | `Boolean` | `false` | 是否显示边框 |
| shadow   | `Boolean` | `false` | 是否显示阴影 |

### LayoutSider Props

| 名称     | 类型      | 默认值  | 描述         |
| -------- | --------- | ------- | ------------ |
| bordered | `Boolean` | `false` | 是否显示边框 |
| boxShadow   | `Boolean` | `true` | 是否显示阴影 |
