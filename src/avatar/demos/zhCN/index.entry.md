# Props

| 名称   | 类型                 | 默认值      | 描述       |
| ------ | -------------------- | ----------- | ---------- |
| src    | `String`             | `undefined` | 图片链接   |
| circle | `Boolean`            | `false`     | 是否为圆形 |
| size   | `Number \| "medium"` | `"medium"`  | 头像大小   |
