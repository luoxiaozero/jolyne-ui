# Props

| 名称     | 类型                                                                    | 默认值     | 描述           |
| -------- | ----------------------------------------------------------------------- | ---------- | -------------- |
| tag      | `String`                                                                | `"button"` | 渲染为什么标签 |
| type     | `"default" \| "primary" \| "error" \| "info" \| "success" \| "warning"` | `default`  | 按钮类型       |
| text     | `Boolean`                                                               | `false`    | 是否为文本按钮 |
| disabled | `Boolean`                                                               | `"false"`  | 是否禁用       |
