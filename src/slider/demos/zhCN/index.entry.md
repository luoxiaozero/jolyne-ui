```demo
basis
delay
```

## Props

| 名称        | 类型      | 默认值     | 描述                 |
| ----------- | --------- | ---------- | -------------------- |
| max         | `Number`  | `100`      | 最大值               |
| step        | `Number`  | `1`        | 步长                 |
| value       | `Number`  | `undefine` | 当前值               |
| sliderDelay | `Boolean` | `false`    | 滑动时是否延迟更新值 |
